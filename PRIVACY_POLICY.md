# Privacy Policy

Draft notice: this file is not the final Privacy Policy.

This public repository does not collect user data by itself.

Before publishing Morsy publicly, this file should be replaced with the final Privacy Policy describing, in clear terms:

- what data the app collects
- why the data is collected
- how account data is used
- whether third-party services are used
- how long data is stored
- how users can request deletion
- contact information for privacy requests

Do not treat this draft as production-ready legal text.
